A Code challenge created for NaturalCircles.

The purpose is to validate the email for typos, to be precise the domains are validated using ngpattern /^[a-z0-9!#$%&'*+/=?^_`{|}~.-]+@gmail\.com|msn\.com|hotmail\.com|live\.com|msn\.com|aol\.com|yahoo\.com$/i

This way we can filter user input and validate it using the correct email domain.
